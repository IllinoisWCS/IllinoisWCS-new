Web Development

For learning the basics of CSS, HTML, and Javascript, we recommend checking out online resources like w3schools or Free Code Camp. Other useful things to learn include frameworks for handling JSON data like JQuery, Moustache, and Angular.

If you’re looking to skip styling your website or are looking for re-useable interfaces, check out Bootstrap or Material Design Lite.

If you’re looking for something more advanced, check out awesome tools and frameworks like React for frontend, Node.js, Flask, or Django for backend. Other awesome frameworks exist! Don’t be afraid to Google for the tools that will meet your needs.

Often, when dealing with backend, you’ll also need a database. SQL and MongoDB are popular languages to learn, and you can find tutorials for these at the links in the first paragraph of this section. Another popular database is Google’s Firebase, which has a free tier if you want to experiment with data.

Lastly, if you’re looking to deploy your website, try out Github Pages (for static sites only), Heroku, or https://www.netlify.com/. GitHub pages also is connected with Jekyll, making it easy for you to deploy your personal site or a project site using only markdown!
