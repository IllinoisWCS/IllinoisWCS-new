What is JSON

  • Javascript Object Notation
  • A way to represent bits of data to be consumed and displayed by a website
  • An object is encased by {} and has properties with values
  • A property can have a boolean, float, int, string or arrays of those primitives as values

The following example shows a JSON object that represents some student data.


{
	"firstname": "Emily",
	"lastname": "Chao",
	"major": "Computer Science",
	"year": 2017,
	"isCrazy": true
}

So how does this fit in with our application?

  • Many data visualization web frameworks use JSON to build their visualizations.
  • JSON objects are easily manipulated - add new attributes, foreach loops based on attributes
  • When your website from your browser wants some data, you make a request to a server. The server's response is JSON.

Example with Javascript

Say your data looks like this:


data = [
	{
		"heartrate": 80,
		"time": "10/27/2017 8:40am"
	},
	{
		"heartrate": 120,
		"time": "10/27/2017 9:50am"
	},
	{
		"heartrate": 125,
		"time": "10/27/2017 10:55am"
	}
];

Example with Javascript cont.

Say you run a function like this:


for (var i = 0; i < data.length; i++) {
	if (data[i].heartrate > 120) {
		data[i].active = true;
	} else {
		data[i].active = false;
	}
}

Example with Javascript cont.

Now your data looks like this in memory.


data = [
	{
		"heartrate": 80,
		"time": "10/27/2017 8:40am",
		"active": false
	},
	{
		"heartrate": 120,
		"time": "10/27/2017 9:50am",
		"active": true
	},
	{
		"heartrate": 125,
		"time": "10/27/2017 10:55am",
		"active": true
	}
];
