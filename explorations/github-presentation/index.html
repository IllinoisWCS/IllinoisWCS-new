What is git?

  • A version control system.
  • Save code in a repository.
  • Can see history of changes - commits.
  • Can collaborate with other people.

What is Github?

  • A website that lets you store git repositories in the cloud.
  • Collaboration tools, issue tracking, wiki - good for teams
  • Can also host static websites.
  • If you like another person's repository, allows you to fork.

How to use git/Github

  • Terminal
  • Github client for your operating system
